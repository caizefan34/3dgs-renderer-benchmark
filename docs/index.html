Why speedy_splat beats diff_gaussian

Sort AlgorithmCUB DeviceRadixSort vs Thrust
Sort Gain+15-30%
Shared MemoryWarp-level coalesced vs standard
Overall Render+1.5%
CUB DeviceRadixSort uses warp-shuffle primitives, avoids intermediate global memory passes, and achieves better occupancy in the tile-binning pipeline.
